# Tele2/Altel Digital Hackaton

Приложение предназначается для бронирования: 

- Рабочих мест
- Переговорных комнат
- Столовых
- Парковок

Должно иметь возможность:

- Создавать объекты бронирования (офисы, парковки, столовые)
- Настраивать максимальный процент посадки (0%-100% от числа возможных мест бронирования)
- Настраивать социальную дистанцию
- Динамически отключать возможность бронирования мест согласно пп. 2-3
- Отображать конфликты бронирования и предлагать другое время.
- Иметь возможность интегрирования: Календарь, Active Directory, Teams и пр.


Улучшить:
- Конфликты в будущем времени
